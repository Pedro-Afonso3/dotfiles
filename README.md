# My Arch Linux + Hyprland Dotfiles

## 📦 Stack
- **WM:** Hyprland
- **Bar:** Waybar
- **Terminal:** Konsole
- **Shell:** ZSH
- **Editor:** Code
- **Launcher:** Rofi
- **Notifications:** Dunst

## 🚀 Installation

```bash
git clone https://github.com/Pedro-Afonso3/dotfiles.git

cd ~/dotfiles/packages
./install-packages.sh

cd dotfiles
./install.sh

sudo reboot
